# Finka Bank — сайт банка (учебный проект)

Статический сайт-витрина банка «Finka Bank»: главная страница с продуктами,
калькулятором кредита, новостями и отзывами, а также страница «Контакты»
с картой и формой обратной связи.

## Структура проекта

```
finka-bank/
├── index.html          Главная страница
├── contacts.html        Страница «Контакты»
├── css/
│   └── styles.css       Все стили (шапка, секции, адаптивность)
├── js/
│   └── script.js        Меню, калькулятор, слайдер, валидация формы
├── images/               Папка для изображений (если будут добавляться)
└── README.md
```

## Как запустить локально

1. Откройте папку `finka-bank` в VS Code.
2. Установите расширение **Live Server** (автор: Ritwick Dey).
3. Откройте `index.html`, нажмите правой кнопкой → **Open with Live Server**.
4. Сайт откроется в браузере по адресу вида `http://127.0.0.1:5500/index.html`.

Никакого backend не требуется — это полностью статический сайт на HTML/CSS/JS.

## Проверка адаптивности (день 03 и 11)

1. Откройте сайт в браузере, нажмите `F12` (DevTools).
2. Включите режим устройства (иконка телефона/планшета или `Ctrl+Shift+M`).
3. Проверьте ширины: **375px** (мобильный), **768px** (планшет), **1280px** (десктоп).

## Калькулятор кредита (день 04)

Формула аннуитетного платежа:

```
платёж = P × (r × (1 + r)^n) / ((1 + r)^n − 1)
```

где `P` — сумма кредита, `r` — месячная ставка (годовая ставка / 100 / 12),
`n` — срок в месяцах. Реализация — в `js/script.js`, функция `calculate()`.
